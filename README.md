# FSB.Application.HRM
Демо-приложение "HR Management System"


Нужно спроектировать демонстрационное консольное приложение на python, имитирующее работу системы автоматизации hr.

Основные функции:
- Зарегистрировать кандидата
- Утвердить кандидата
- Отклонить кандидата

Технологический стек:
- typer
- sqlite
- pydantic
